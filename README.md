# 🛒 Web Scraping de E-commerce

Este é um script Python para coletar informações de um site "que nao existe"

Devo destacar que minha primeira vez testando e aprendendo como funciona web scraping, por mais que conheca o proposito na praticava pensava que era mais facil
a dificuldade esta no processo de verificar como conseguir o arquivo que chega ate o site e depois coletar e entender a estrutura para que ai sim, o script tenha alguma relevancia,
nao era tao simples como imaginei, nao e so ter um scrip e preencher alguns dados a um trabalho para se entender a estrutura o que levou algumas horas, mas foi divertido.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **Requests:** Para envio de requisições HTTP e download do HTML.
* **BeautifulSoup4:** Para análise (parsing) do HTML e extração das tags de script.
* **JSON & CSV:** Bibliotecas nativas do Python para manipulação de objetos e exportação dos dados.
